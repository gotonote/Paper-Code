# MedRAX: Medical Reasoning Agent for Chest X-ray

## 论文信息
- **会议**: ICML 2025
- **Stars**: 1099
- **作者**: bowang-lab

## 核心内容

MedRAX 是一个用于胸部 X 光片医学推理的智能体框架。该论文提出了一种新颖的医学多模态大语言模型方法，能够对 X 光图像进行自动诊断推理。

### 主要贡献
1. **医学推理智能体**: 构建了专门用于医学影像分析的智能体系统
2. **胸部 X 光分析**: 能够对胸部 X 光片进行全面的疾病诊断
3. **推理链生成**: 生成详细的医学推理过程，帮助医生理解诊断依据

### 技术亮点
- 多模态融合技术，结合图像和文本信息
- 医学领域知识图谱的运用
- 可解释的诊断推理过程
