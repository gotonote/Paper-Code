## 核心内容

本文提出了 **MindJourney**，一种利用世界模型进行测试时扩展的空间推理方法。

### 主要贡献

1. **世界模型**: 构建能够模拟物理世界的内部模型。

2. **测试时扩展**: 在推理阶段通过增加计算资源提升模型性能。

3. **空间推理**: 专注于空间理解和推理能力的提升。

### 技术细节

- **世界模型架构**: 模拟空间关系的神经网络
- **推理策略**: 多步推理和路径规划
- **评估基准**: 专门的空间推理评估任务
