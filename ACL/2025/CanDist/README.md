# CanDist: Context-Aware Distance Metric Learning

> ACL 2025

## 论文信息

- **标题**: CanDist: Context-Aware Distance Metric Learning for NLP
- **作者**: MingxuanXia
- **链接**: [GitHub](https://github.com/MingxuanXia/CanDist)

## 核心贡献

1. 提出上下文感知距离度量学习方法
2. 适用于各种NLP任务
3. 在文本相似度和语义匹配上取得优异表现

## 方法概述

CanDist通过学习上下文感知的距离度量，更好地捕捉文本间的语义关系。

## 运行方式

```bash
pip install -r requirements.txt
python train.py --dataset banking
```

## 依赖

- PyTorch >= 1.10
- transformers
- numpy
