# 鲁棒性重编程：表征学习的新方向

## 论文概述

本研究探索了一个有趣且具有根本性意义的问题：给定一个训练良好的深度学习模型，能否在不改变其参数的情况下，通过重编程来增强其对抗对抗性或噪声输入扰动的鲁棒性？

## 核心内容

### 主要贡献

1. **新型鲁棒模式匹配技术**：重新审视表征学习中的核心特征转换机制，提出了一种新颖的非线性鲁棒模式匹配技术

2. **三种模型重编程范式**：提供了三种在 不同效率要求下灵活控制鲁棒性的范式：
   - **范式 1**：无需微调，零成本获得良好鲁棒性
   - **范式 2**：仅微调 λ 参数，轻微成本获得强鲁棒性
   - **范式 3**：全面微调，可接受成本获得卓越鲁棒性

3. **广泛验证**：在从基本线性模型、MLP、浅层和现代深度卷积神经网络等多样化的学习模型上进行了全面实验

### 技术意义

- 为深度学习中的对抗防御提供了有前景且正交的新方向
- 为设计更具弹性的 AI 系统提供了新见解
