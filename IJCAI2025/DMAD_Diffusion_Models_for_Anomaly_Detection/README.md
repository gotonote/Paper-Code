# DMAD: 异常检测扩散模型综述

## 核心内容

这是一篇关于**异常检测扩散模型**的综述论文，系统性地分析了扩散模型在异常检测领域的最新进展。

### 主要内容

1. **基础概念**：从基础概念出发，系统介绍经典扩散模型架构（DDPMs、DDIMs、Score SDEs）

2. **方法分类**：将现有 DMAD 方法分为三大类：
   - **基于重建的方法** (Reconstruction-based)
   - **基于密度的方法** (Density-based)
   - **混合方法** (Hybrid)

3. **数据模态**：涵盖多种数据模态的应用：
   - 图像异常检测
   - 时间序列异常检测
   - 视频异常检测
   - 多模态异常检测

4. **关键挑战**：讨论了以下关键挑战：
   - 计算效率
   - 模型可解释性
   - 鲁棒性增强
   - 边缘云协作
   - 与大语言模型的集成
