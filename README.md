## 🎯 内容包含

每个论文目录包含：

- **README.md** - 论文核心内容讲解（中文）
  - 论文标题、作者、链接
  - 核心贡献总结
  - 方法概述
  - 代码结构说明
  - 运行方式

- **核心代码** - 论文核心算法实现

- **依赖文件** - requirements.txt / environment.yml

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/gotonote/Paper-Code.git

# 进入目录
cd Paper-Code/CVPR/2025/3D_Gaussian_Splatting

# 查看论文解读
cat README.md

# 安装依赖
pip install -r requirements.txt
```
