# Neural Radiance Field (NeRF) - CVPR 2025

## 论文信息

- **标题**: NeRF in the Wild: Neural Radiance Fields for Unconstrained Photo Collections
- **作者**: Ricardo Martin-Brualla et al.
- **链接**: https://nerf-w.github.io/
- **会议**: CVPR 2025

## 核心贡献总结

1. 提出针对野外照片的 NeRF 变体
2. 引入外观嵌入实现光照一致性
3. 使用图像质量检测器处理低质量输入
4. 实现鲁棒的新视角合成
