# UP-VLA: Unified Perception VLA

> ICML 2025

## 论文信息

- **标题**: UP-VLA: Unified Perception Vision-Language-Action Model
- **作者**: CladernyJorn
- **链接**: [GitHub](https://github.com/CladernyJorn/UP-VLA)

## 核心贡献

1. 提出统一的感知型视觉-语言-动作模型
2. 简化了VLA架构设计
3. 在机器人操作任务上取得优异性能

## 方法概述

UP-VLA通过统一感知模块，实现了更高效的多模态学习。

## 运行方式

```bash
pip install -r requirements.txt
python train.py --task push
```

## 依赖

- PyTorch >= 2.0
- numpy
- gym
